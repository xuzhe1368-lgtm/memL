# memL v0.3 路线图（稳健生产版）

## 目标
将 memL 从“可用”升级为“可持续运维”：可备份恢复、可观测告警、可灰度发布、可解释检索。

## M1（优先）数据安全与可恢复
- [ ] 增加 tenant 级导出脚本（JSONL）
- [ ] 增加 tenant 级导入脚本（JSONL）
- [ ] 增加备份校验与恢复演练文档
- [ ] 增加最小恢复验收流程（RTO/RPO 基线）

## M2 检索质量
- [ ] 时间衰减打分（可配置）
- [ ] 标签权重（可配置）
- [ ] 返回 match explain（向量分/关键词分/标签分）

## M3 运维与发布控制
- [ ] Prometheus 指标格式输出
- [ ] 告警阈值模板（错误率、embedding 失败率、P95）
- [ ] tenant 级灰度开关（新检索策略）
- [ ] 一键回滚脚本增强（保留最近 N 份）

## 发布节奏
- v0.3.0-alpha: M1
- v0.3.0-rc1: M2
- v0.3.0: M3 + 回归
